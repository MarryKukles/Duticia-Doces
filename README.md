## 🧁 Sobre o Projeto

**Dutícia Doces** é um site web desenvolvido para registrar e avaliar doces que eu e minha noiva experimentamos. Permite salvar informações como nome, data, valor pago, foto, comentários e nota (0 a 10), com avaliação visual em estrelas — incluindo **meias estrelas** para maior precisão.

O sistema utiliza **Supabase** como backend (banco, storage e realtime) e **Next.js** no frontend, garantindo uma experiência rápida, moderna e responsiva.

---

## ✨ Principais Funcionalidades

- 📝 **Cadastro completo de doces** (nome, data, valor, nota, comentário e foto)
- ⭐ **Avaliação com estrelas** (suporte a meia estrela)
- 📸 **Upload de imagens** direto para o Supabase Storage
- ⚡ **Sincronização em tempo real** entre dispositivos
- 🔍 **Busca e filtro** por nome, valor ou comentário
- 🧩 **Edição e exclusão** de doces existentes
- 📱 **Layout responsivo** para desktop e mobile
